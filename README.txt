# Website Enjoy Baturaden

Website Enjoy Baturaden. Untuk frontend menggunakan Bootstrap v4.5.0 dan untuk backend menggunakan PHP Native.

- Pengkoneksian ke dalam database MySQL 
  $db['default']['hostname'] = 'localhost';
  $db['default']['username'] = 'root';
  $db['default']['password'] = '';
  $db['default']['database'] = 'serkom';

  $koneksi= new mysqli('localhost','root','','nama-database');

Minimum requirements :
- PHP 5.x
- MySql 5.x atau MariaDB 5.x

Elements :
- Bootstrap v4.5.0
- Chart

Style :
- Menggunakan Media Query, yang digunakan untuk mengatur tampilan atau style website agar responsif dengan teknik CSS (Cascading Style Sheets) yang mengacu pada CSS3.
